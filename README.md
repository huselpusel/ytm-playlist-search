# 🎵 YTM Playlist Search

![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

A lightweight, local-only Chrome extension that allows you to instantly search through all songs in a YouTube Music playlist without endlessly scrolling.

</p>

## ✨ Features
* **Zero Data Exfiltration:** Runs completely locally in your browser. No user data is collected.
* **Instant Background Loading:** Fetches the entire playlist seamlessly via internal InnerTube endpoints.
* **Dual Action Mode:** Choose between scrolling directly to the song or playing it instantly.
<p align="center">
  <img src="Settings%20menu.png" width="300">
</p>

* **Native UI:** Blends perfectly with YouTube Music's dark mode.

## 🚀 Installation (Developer Mode)
1. Go to the **Releases** section on the right side of this page and download the `YTM-Playlist-Search-v1.0.zip` file.
2. Extract the ZIP file to a permanent location on your computer.
3. Open Chrome and navigate to `chrome://extensions/`.
4. Enable **Developer mode** (top right corner).
5. Click **Load unpacked** and select the extracted folder.
